# Project-1


# I-70 SUCKS


## What?
A delightful game for all ages that allows the player to relive over and over the never-ending terror of I-70 durring ski season.  

## How? 
By using the "<-" and "->" keys in single player mode and adding the "a" and "d" in two player mode, players move their vehical from lane to lane attempting to avoid obsticls such as slow drivers and Texans.
